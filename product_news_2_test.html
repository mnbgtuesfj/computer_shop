CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器實測 最強賊船冷卻旗艦出航


2021-07-20 9:26

CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器。



在追求極限效能的領域中，處理器與顯示卡的散熱一向都是玩家最注重的部分，怎麼樣才不會讓系統在執行過程中因過熱而導致效能驟降，來確保順暢的遊戲效能，大型的塔型空冷散熱器、一體式水冷、甚至是複雜的分體水冷等配置，在這幾年幾乎是高階玩家的基本配備。而這次陳拔要來開箱測試的，是 CORSAIR 最新推出的 H170i ELITE CAPELLIX AIO CPU 水冷散熱器，以三具 140mm 風扇搭配 420mm 的水冷排，是目前市面上一體式水冷散熱器中，尺寸最大的款式，而實際上的散熱表現如何？以下是陳拔的開箱報告。



CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器開箱與外觀



首先來看 CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器的開箱與外觀說明。





CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器的外盒正面，跟去年陳拔開箱的 H115i ELITE CAPELLIX AIO CPU 水冷散熱器是同一系列（請見：CORSAIR iCUE H115i iCUE ELITE CAPELLIX 280 AIO 水冷實測 超亮燈效設計搭配強力散熱），所以在外盒設計上都是採用 CORSAIR 近年來的黃黑配色。





外盒標註了為 420mm 規格的水冷處理器散熱器，並且具有 RGB 燈效。



採用 CORSAIR 新款的 CAPELLIX 超亮 LED 燈效設計，可搭配 CORSAIR 自家的 iCUE 軟體進行控制。



外盒上方側面的右上角有 CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器盒內的配件圖示。



左上角則有支援的處理器插槽類型以及支援的處理器，作業系統則是支援 Windows 10。



CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器的盒裝背面，以實際影像呈現產品的特色。



中央部分則是標示了水冷排安裝風扇後的尺寸，需要長度 457mm、寬度 140mm、高度 52mm 的空間進行安裝。



左側則是以實際的影像呈現 CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器安裝兩款不同水冷頭蓋板後的燈效顯示情形。



右側則是以多國語言更詳細說明 CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器的尺寸規格。



CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器的盒裝內容物，由上到下、由左至右分別為：三具 140mm RGB 風扇、保固說明、 420mm AIO 水冷排、安裝指南、透明水冷頭飾板、風扇控制器（藍色袋內）、AMD AM4 插槽安裝配件、AMD TR4 插槽安裝配件、intel 插槽安裝配件、螺絲配件包等。



AIO 水冷排本體， 採用 420mm 規格真的相當巨大。



冷排尺寸與 CORSAIR H115i ELITE CAPELLIX 280 AIO CPU 水冷散熱器（下）相比。



冷排尺寸與 CORSAIR iCUE H150i RGB PRO XT 360 AIO CPU 水冷散熱器（下）相比。



水冷排本身採用14 排散熱格柵設計。



上方一貫具有 CORSAIR 海盜船的圖騰徽飾。



水冷排與進出水管的連接設計，水管外部則以編織材質包覆，水管長度則為 450 mm。



CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器的水冷頭配置，採用跟 H115i ELITE CAPELLIX AIO 相同的設計，內部具有 33 顆超亮 CAPELLIX RGB LED 燈珠，提供更亮的水冷頭燈效顯示。



未安裝飾蓋前水冷頭的燈效顯示狀況。



安裝預設的黑色飾蓋時，水冷頭的燈效顯示狀況。



而這次 CORSAIR 也一樣的提供了另一組可替換的白色半透明水冷頭飾蓋，可以讓使用者在兩種不同燈效呈現模式中選擇。



CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器盒裝配件內的白色半透明水冷頭飾蓋。



使用白色半透明水冷頭飾蓋時的燈效顯示狀況。



CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器的水冷頭底部，採用 CORSAIR 的分流式銅質導熱板設計，表面採用每英吋128 條的微切翅片堆疊設計，提供在高負載下的快速導熱性能。另外也可見到表面已經預塗了散熱膏。



相較於近期市面上頂級的 AIO 水冷，在水冷頭部分均採用大型的內崁式風扇設計，不具內崁風扇的 CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器水冷頭體型相對小巧，水冷頭高度大約在 50mm 左右。



另外由於 CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器這次也配備了獨立風扇控制器的緣故，所以水冷頭部分並未有 4 Pin 風扇連接線，而是以風扇控制器的傳輸線取代，另外一條則是連接主機板 CPU FAN 插槽的 3 Pin 連接線。



CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器水冷頭水管連接處的設置，採用 90º 轉角可轉動的設計。



CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器配屬的風扇，採用磁懸浮設計，轉速範圍為 400 至 2000 RPM、單一風扇氣流量為 97 CFM、風壓為 0.2 至 3. mm-H2O，風扇噪音值為 10 至 36 dB(A)。



CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器配屬的風扇，採用磁懸浮設計，轉速範圍為 400 至 2000 RPM、單一風扇氣流量為 97 CFM、風壓為 0.2 至 3. mm-H2O，風扇噪音值為 10 至 36 dB(A)。



風扇的另一面，另外可看到具有 4 Pin 電源連接線以及 ARGB 燈效連接線各一組的配置。



實際開機時的風扇燈效顯示狀況。



在全黑環境下拍攝，可以看到在風扇軸心外側有 8 顆 LED 燈珠分佈。



CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器所配屬的風扇控制器，跟 H115i ELITE CAPELLIX AIO CPU 水冷散熱器的是相同的，提供了六組風扇的電源以及燈效控制。



風扇電源插座部分。



風扇控制器的 RGB 燈效連接的部分。



前方則是具有與水冷頭傳輸線連接的介面。



另一邊則是設置了不可更換的電源線以及主機板傳輸線部分，CORSAIR 也在這一側設置了溫度感應線的接點，讓使用者可以在這裡接上溫度感應線。



風扇控制器的電源供應則是直接以 SATA 接頭提供，需要與電源供應器的 SATA 電源線連接，另外一條則是連接主機板 USB 2.0 連接埠的傳輸線。



用獨立風扇控制器的設計的確在裝機時可以省下在水冷頭處煩雜的整線設計，不過可惜的是這次 CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器的風扇控制器仍是以搭配膠條的方式固定在機殼上，或許可以採用類似 ROG RYUJIN II 360 AIO 水冷散熱器的風扇控制器設計（請見：ROG RYUJIN II 360 AIO 水冷散熱器實測 將敗家信仰提升到最高點），設計為 2.5 吋磁碟機的大小，並且提供固定螺絲孔，讓使用者可以彈性的方式固定在機殼背面上



CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器軟體控制



CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器仍是使用自家 iCUE 軟體進行控制，提供包括燈效、風扇轉速、水幫浦轉速、運作模式、韌體更新等功能。


iCUE 軟體主界面，使用目前最新的 4.13.223 版本。



點擊『H170i ELITE CAPELLIX』圖示後就可以進入 CORSAIR H170i ELITE CAPELLIX AIO CPU 水冷散熱器的控制介面，首頁顯示包括水幫浦（泵）、 三具風扇的燈效顯示情形。



若是有將其他機殼風扇連接到風扇控制器，也可以在這個界面進行操控，在左下角可以指定風扇的類型。



進入『燈光效果』頁面後則是可以針對個別組件的燈光效果進行調整。



提供包括彩虹波動、螺旋彩虹、彩虹、黑色偏移、顏色脈動、顏色流動、依序、頻閃、雨水等燈光效果，另外也可分別指定各組件的燈效以及調整燈效變換速度。



在『冷卻中』選項裡，則是可以調整風扇與水冷頭幫浦（泵）的運轉模式，風扇部分分為安靜、均衡、極限、零 RPM 四種模式，水冷頭幫浦（泵）的部分則是將零 RPM 的部分更換為可變速度選項。另外也可以在這裡看到冷卻液目前的溫度。



除了可以全部套用同樣模式外，也可針對個別組件指定運作模式。



另外也可將組件重新命名，或是新增到主面板方便管理。



點擊個別組件時，下方還會出現詳細的運作狀態，圖為風扇部分過去一段時間運作的轉速。



也可以監控水冷液的溫度。



『警告』部分則是會依照設定的條件（如水冷液溫度、風扇轉速）提出警示，也可設定若超過設定值時會自動關機保護的機制。



『裝置設定』部分則是提供包括韌體更新、整體 LED 亮度調整的功能。
